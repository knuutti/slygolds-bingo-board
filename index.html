Rules

You know how Bingo works. These rules just clarify some edge cases.

  1. An item counts as located as soon as we know where it is. Runners don't need to collect the item. For example, if a racer sees a hint indicating where Nayru's Love is, you're allowed to mark it on your board. Likewise if they peek it.
  2. You alone are responsible for identifying when you need to check off a square. If you miss one, too bad.
  3. If you report a false Bingo, you must sing a song.
  4. Not all Bingo cards are created equal, that's the nature of a random number generator.
  5. Most importantly: Have fun!
